APK2AAB Converter
=================

This is a universal package that works on both Windows and macOS.

Instructions:

Windows:
1. Double-click APK2AAB.bat to run the application

macOS:
1. Open Terminal in this directory
2. Run: chmod +x APK2AAB.sh
3. Run: ./APK2AAB.sh

Using the Application:
1. Select your APK file and choose where to save the AAB
2. Click Convert to start the conversion process

Requirements:
- Java 11 or later must be installed
